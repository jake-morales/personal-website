USC Volunteer Technical Staff

USC Dornsife Mind & Society Center — Sep 2024 – Feb 2025

Overview

Volunteered as technical staff for the USC Dornsife Mind & Society Center, a research center focused on social and behavioral science. Provided infrastructure support to graduate students whose data analysis workflows were constrained by the limits of personal hardware.

Set up self-hosted data tools and cloud compute resources to unblock research work that would otherwise have been impossible or impractical on standard laptops.

Tech Stack

Metabase PostgreSQL RStudio Server AWS EC2

Key Contributions

  • Enabled graduate students to perform data analysis and visualization by setting up Metabase connected to their Postgres database
  • Deployed RStudio Server on a high-memory AWS EC2 instance, resolving memory constraints on graduate student laptops from resource-intensive analyses
