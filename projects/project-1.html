Project One

A task management app

Overview

This is a full-stack task management application designed to help teams organize their work efficiently. It features real-time updates so that all team members see changes as they happen, without needing to refresh the page.

The project was born out of frustration with existing tools that were either too complex or too simple. The goal was to build something that strikes the right balance between power and usability.

Users can create projects, assign tasks, set deadlines, and track progress through a clean, intuitive interface. The app supports drag-and-drop reordering and real-time notifications.

Tech Stack

React Node.js Express PostgreSQL Socket.io Docker

Key Features

  • Real-time collaboration with WebSocket connections
  • Drag-and-drop task reordering
  • User authentication and role-based permissions
  • Email notifications for task assignments and deadlines
  • REST API with comprehensive documentation
