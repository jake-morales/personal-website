Project Two

A weather dashboard

Overview

A responsive weather dashboard that pulls data from a public weather API to display current conditions, hourly forecasts, and 7-day outlooks for any location worldwide.

The project started as a way to learn API integration and data visualization. It evolved into a polished tool that I use daily to check conditions before heading out.

The dashboard supports location search with autocomplete, saves favorite locations, and displays historical weather data with interactive charts.

Tech Stack

JavaScript Chart.js OpenWeather API Bulma CSS LocalStorage

Key Features

  • Current weather, hourly, and 7-day forecasts
  • Location search with autocomplete suggestions
  • Interactive charts for temperature and precipitation
  • Save and manage favorite locations
  • Responsive design that works on mobile and desktop
